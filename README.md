# Sorry! Board Game (Java)

## Description
Java implementation of the board game **Sorry!**, developed as part of the *CS252 - Object-Oriented Programming* course at the University of Crete.

The project follows the **MVC (Model–View–Controller)** architecture and includes a graphical user interface.

---

## Features
- Turn-based gameplay
- Card-based movement system
- Full game logic (slides, safety zones, collisions, home)
- Java Swing GUI

---

## Structure
- **Model**: Game logic (cards, players, deck, pawns)
- **View**: GUI
- **Controller**: Game flow and interaction

---

## Run
1. Open the project 
2. Navigate to: `src/Main/main.java`  
3. Open the file  
4. Click the **"Run" button** above the `main` method
